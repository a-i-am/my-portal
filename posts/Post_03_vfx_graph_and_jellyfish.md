## 1. 수많은 해파리를 화면에 그리는 방법

미디어 아트에서는 화면을 꽉 채우는 수많은 빛 입자(파티클)나 생물들의 움직임이 중요합니다. 'Jellyfish Dancers' 프로젝트에서는 수십 마리의 해파리가 부드럽게 헤엄치는 모습을 표현해야 했습니다. 

일반적인 방식으로 컴퓨터에게 해파리 한 마리 한 마리의 움직임과 형태를 계산하라고 명령하면 컴퓨터가 너무 느려집니다. 그래서 유니티(Unity)의 **Visual Effect Graph (VFX Graph)** 라는 기능을 사용했습니다. 이 기능은 그래픽 카드(GPU)를 이용해 수만 개의 입자를 한 번에 매우 빠르게 그려낼 수 있습니다.

## 2. 해파리 움직임 제어하기 (JellyfishAgent)

해파리의 기본 뼈대와 움직임 규칙은 `JellyfishAgent.cs` 라는 파이썬이나 C# 코드로 작성했습니다. 해파리가 물속에서 헤엄치는 것처럼 보이게 만들기 위해 여러 가지 물리 법칙을 코드로 구현했습니다.

- **헤엄치기(Propulsion)**: 해파리가 몸통을 수축했다가 이완하면서 앞으로 나아가는 힘을 계산합니다.
- **물살(Ocean Current)**: 보이지 않는 물결이 밀려오는 것처럼, 약간씩 방향을 틀며 떠다니도록 만듭니다.
- **충돌 방지(Separation)**: 다른 해파리들과 너무 가까워지면 서로 밀어내어 부딪히지 않게 합니다.

```csharp
// _Project/Scripts/VFX/JellyfishAgent.cs 내부 움직임 제어 코드의 일부

// 1. 관객의 움직임(모션) 정도에 따라 해파리의 흥분도(agitation)를 계산합니다.
float agitation = 1f + motionScore * 2f;

// 2. 해파리가 헤엄치는 주기(수축과 이완)를 계산합니다.
EvaluateSwimCycle(agitation, out float contraction, out float propulsion);

// 3. 앞으로 나아가는 힘을 계산합니다. 흥분도가 높을수록 더 강하게 헤엄칩니다.
Vector3 propulsionForce = transform.up * propulsion * (powerStrokeForce * agitation);

// 4. 관객이 있는 위치나 화면 중앙을 향해 끌려가는 힘(Attractor)을 계산합니다.
Vector3 interactionForce = CalculateAttractionForce(attractorWorldPos);

// 5. 물살의 힘과 헤엄치는 힘, 끌려가는 힘을 모두 더해서 현재 속도에 반영합니다.
currentVelocity += (propulsionForce + oceanCurrent + interactionForce) * Time.deltaTime;
```

## 3. 관객과 상호작용하는 해파리 촉수

해파리의 몸통은 3D 모델로 만들었지만, 아래로 길게 늘어지는 수십 개의 촉수는 코드로 선(Line)을 그려서 표현했습니다. 

촉수가 딱딱한 막대기처럼 보이지 않고 물속에서 부드럽게 찰랑거리도록 만들기 위해, 촉수를 여러 마디로 나누고 앞 마디가 움직이면 뒷 마디가 고무줄처럼 약간 늦게 따라오도록(Verlet Integration 물리 공식) 설정했습니다.

여기에 앞서 [제1편]에서 설명했던 스마트폰 터치 신호나, [제2편]에서 뽑아낸 관객의 실루엣 정보를 연결합니다. 관객이 화면 근처로 다가오거나 스마트폰 화면을 세게 누르면, 그 위치로 해파리들이 일제히 방향을 틀고 촉수를 뻗도록 상호작용(Interaction)을 만들었습니다.

## 4. 시각적 효과 극대화

이렇게 계산된 해파리들의 위치 정보와 관객의 실루엣 정보(흑백 마스크)는 최종적으로 VFX Graph에 전달됩니다. 

VFX Graph는 이 정보들을 바탕으로:
1. 관객의 실루엣 주변에 수천 개의 반짝이는 입자를 뿌립니다.
2. 해파리가 헤엄칠 때 몸통에서 빛의 궤적이 남도록 그립니다.
3. 배경음악의 박자에 맞춰 해파리의 색깔과 밝기를 조절합니다.

코드는 해파리가 "어디로 움직일지"만 계산하고, 반짝이는 시각적 효과는 그래픽 카드(VFX Graph)가 전담하여 처리하기 때문에 수백 마리의 해파리가 나와도 부드러운 화면(60fps 이상)을 유지할 수 있습니다.
