# WHERE IS MY TOILLET: 게임잼 2D 플랫포머 개발기

이 시리즈는 인하대학교 게임잼에서 3일이라는 짧은 시간 동안 2D 플랫포머 게임을 개발하며 겪었던 **레벨 디자인(맵), 2D 애니메이션(Spine), 그리고 적 AI(인공지능) 구현 과정**을 다룹니다.

## 📝 시리즈 목차 (Draft)

- **[제1편] 2D 맵 설계: Tilemap을 활용한 빠른 레벨 디자인**
  - 짧은 게임잼 기간 내에 2D 플랫포머 맵(Forest 맵 등)을 타일맵(Tilemap)으로 빠르게 구성한 방법
  - 맵 제작 과정에서 겪은 충돌 처리와 타일 최적화 경험
- **[제2편] 애니메이션 제어: Spine과 Unity Animator의 만남**
  - 유니티 기본 애니메이터(Animator) 대신 스파인(Spine)을 활용하여 캐릭터 애니메이션을 적용한 방법
  - `SpineAnimatorBehaviour`를 통해 스파인의 뼈대와 유니티의 상태 머신을 연결하여 부드러운 모션을 제어하는 원리
- **[제3편] 적 AI 구현: FSM 기반의 똑똑한 몬스터 만들기**
  - 복잡한 몬스터의 행동을 탐지, 순찰(Patrol), 추적(Chase) 상태로 나누어 제어하는 FSM(유한 상태 기계) 구조 도입기
  - 몬스터 행동 제어: 순찰과 추격 상태 구현 (상태 전환 로직과 규칙 분리)
  - 플레이어 감지 최적화와 시야 처리 (시야 감지 범위와 코드 분리)

---

## 📌 주요 기술 스택
- Unity 2022.3.x (2D)
- Tilemap System
- Spine 2D Animation
- FSM (Finite State Machine) 기반 인공지능 패턴
