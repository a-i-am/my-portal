# YOU: 2D 캐주얼 액션 데모 개발기

이 시리즈는 1인 개발로 진행된 2D 캐주얼 액션 데모 프로젝트 **'YOU'**의 핵심 시스템 구현 과정을 다룹니다.
적을 기절시켜 수집하고 아군 팔로워(Follower)로 소환하는 독특한 전투 시스템과, 다수의 팔로워를 효과적으로 제어하기 위한 알고리즘, 인벤토리 구조 설계, 그리고 Cinemachine과 DOTween을 활용한 연출 기법을 정리했습니다.

## 📝 시리즈 목차 (Draft)

- **[제1편] 적을 아군으로: 포획과 팔로워 소환 파이프라인**
  - 적을 기절시키고 수집(포획)하여 인벤토리에 등록하기까지의 기본 흐름 구현
  - 수집한 적을 내 뒤를 졸졸 따라다니는 아군(Follower)으로 다시 소환하는 시스템 구조
- **[제2편] 다중 팔로워 제어: 똑똑한 타겟팅과 공격 로직**
  - 여러 마리의 팔로워가 하나의 적을 중복해서 공격하지 않도록 막는 거리 기반 고유 타겟팅 알고리즘
  - 적을 향해 돌진(대시 공격)하고 다시 원래 내 뒤(기존 위치)로 돌아오는 복귀 로직
- **[제3편] 데이터의 분리: 유연한 인벤토리 리팩터링**
  - 게임의 원본 아이템 데이터와, 슬롯에 장착될 때마다 생성되는 인스턴스(Instance) 데이터를 분리한 이유
  - 유지보수와 확장성을 높인 인벤토리 구조 개선 과정
- **[제4편] 게임에 생동감 불어넣기: Cinemachine과 DOTween**
  - 시네머신(Cinemachine)을 활용한 흔들림 없는 카메라 워킹과 보스전 패턴 연출
  - DOTween을 이용해 UI와 컷신에 부드러운 트윈(Tween) 애니메이션을 입히는 최적화 방법
