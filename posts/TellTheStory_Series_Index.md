# Tell The Story: 음성 인식 기반 3D 연기 시뮬레이션 개발기

이 시리즈는 컴투스 글로벌 게임개발 공모전 출품작인 **'Tell The Story'** 프로젝트의 개발 과정을 다룹니다.
플레이어가 직접 목소리로 대사를 연기하며 이야기를 만들어가는 독특한 컨셉의 게임으로, 3D 공간 속 2D 빌보드 캐릭터 제어, 음성 인식(STT/TTS) API 연동, 대화형 FSM 설계, 그리고 유니티 에디터 커스텀 툴 제작 경험을 정리했습니다.

## 📝 시리즈 목차 (Draft)

- **[제1편] 3D 공간 속의 2D: 빌보드 기법과 8방향 스프라이트**
  - 3D 배경 안에서 2D 캐릭터가 카메라를 항상 바라보게 만드는 빌보드(Billboard) 기법 구현
  - 카메라의 위치와 캐릭터가 바라보는 각도에 따라 8방향 스프라이트 이미지를 부드럽게 전환하는 로직
- **[제2편] NPC와의 상호작용: FSM을 활용한 대화 플로우 제어**
  - 로비에서의 캐릭터 상호작용과 대화 시스템을 유한 상태 기계(FSM)로 깔끔하게 설계한 경험
  - UI 화면의 전환(플로우)과 카메라(Cinemachine) 연출을 자연스럽게 이어주는 제어 구조
- **[제3편] 목소리로 연기하다: NAVER CLOVA API 연동기**
  - 플레이어의 목소리를 텍스트로 변환(STT)하고, 텍스트를 NPC의 음성으로 출력(TTS)하는 네이버 클로바 API 연동 과정
  - API 자격증명(Client ID, Secret)을 소스코드에서 숨기고 안전하게 관리하는 방법
- **[제4편] 반복 작업 줄이기: TMP 폰트 일괄 교체 커스텀 에디터 툴**
  - 수십 개의 씬과 UI 프리팹에 적용된 TextMeshPro 폰트를 하나하나 수작업으로 바꾸는 고통에서 벗어나기
  - 버튼 클릭 한 번으로 프로젝트 전체의 TMP 폰트를 일괄 교체해 주는 나만의 유니티 에디터 툴 제작기

---

## 📌 주요 기술 스택
- Unity 2022.3.x (3D)
- Cinemachine
- NAVER CLOVA Speech API (STT/TTS)
- Custom Editor (Editor Window)
