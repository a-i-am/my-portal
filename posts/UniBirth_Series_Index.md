# Uni Birth: 언리얼 엔진 5 기반 반응형 턴제 RPG 개발기

이 시리즈는 언리얼 엔진 5(UE 5.5)를 기반으로 개발한 턴제 RPG **'Uni Birth'** 프로젝트에서 제가 담당한 주요 시스템들의 구현 경험을 다룹니다.
단순히 공격과 방어를 주고받는 전통적인 턴제 전투를 넘어, 실시간 반응(회피/패링 QTE)과 복잡한 버프 합성 로직을 통해 전투에 긴장감을 불어넣고자 한 고민의 흔적들을 담았습니다.

## 📝 시리즈 목차 (Draft)

- **[제1편] 실시간 반응형 전투: 회피와 패링 QTE 시스템 설계**
  - 적의 공격 턴에도 플레이어가 개입할 수 있도록 만든 QTE(Quick Time Event) 시스템의 기본 설계
  - 정확한 타이밍에 버튼을 입력했는지 판정하는 로직과 데이터 테이블(CSV)을 연동하여 몬스터별로 타이밍을 쉽게 조절한 방법
- **[제2편] 상태 변화의 중심: 버프/디버프 시스템과 상태 동기화**
  - 단순한 스탯 증감을 넘어 턴 경과에 따라 자동으로 해제되거나 중첩되는 버프/디버프의 관리 로직
  - 콤보 성공 및 실패에 따라 UI의 버프 슬롯 상태를 실시간으로 동기화한 과정
- **[제3편] 전략적 깊이 더하기: 동일 등급 버프의 연쇄 합성 시스템**
  - 같은 등급의 버프가 여러 개 모였을 때 자동으로 상위 등급의 버프로 합성되는 '연쇄 합성 시스템'의 구현 원리
  - 합성 과정에서 발생할 수 있는 데이터 누락이나 타이밍 이슈를 해결한 방법

---

## 📌 주요 기술 스택
- Unreal Engine 5.5.4
- C++ Scripting
- UMG (Unreal Motion Graphics)
- Data Table (CSV 연동)
