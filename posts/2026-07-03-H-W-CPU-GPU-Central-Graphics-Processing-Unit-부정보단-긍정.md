# [H/W] CPU & GPU(Central/Graphics Processing Unit)

<h3 data-ke-size="size23"><b>CPU는 중앙처리장치(Central Processing Unit)로 컴퓨터의 두뇌 역할,</b></h3>
<div style="background-color: #ffffff; color: #444444; text-align: start;">
<div style="color: #000000;">
<h3 style="list-style-type: none;" data-ke-size="size23"><b>GPU는 그래픽처리장치(Graphics Processing Unit)로 대량의 병렬 연산에 특화된 장치</b></h3>
<p data-ke-size="size16">&nbsp;</p>
<h2 style="list-style-type: none;" data-ke-size="size26">CPU (Central Processing Unit, 중앙처리장치)</h2>
<p style="list-style-type: none; color: #000000;" data-ke-size="size16">CPU는 컴퓨터의<span>&nbsp;</span><b>중앙 연산 장치</b>로, 프로그램 명령을 해석하고 실행하며 시스템을 제어하는 역할을 합니다<span>&nbsp;</span><span style="background-color: #ffffff;" data-priority="2" aria-hidden="true"></span></p>
<ul style="list-style-type: disc;" data-ke-list-type="disc">
<li style="list-style-type: disc;"><b>주요 특징</b>: 소수의 고성능 코어(보통 2~16개)로 구성되어 복잡한 논리 연산과 다양한 명령어를 빠르게 처리.</li>
<li style="list-style-type: disc;"><b>활용 분야</b>: 웹 브라우징, 문서 작업, 운영체제 실행 등 일반적인 컴퓨터 작업에 필수적.</li>
<li style="list-style-type: disc;"><b>연산 방식</b>: 정수 연산과 부동소수점 연산을 모두 수행하며, 복잡한 계산과 순차적 명령 처리에 강점.</li>
</ul>
<p data-ke-size="size16">&nbsp;</p>
<h2 style="list-style-type: none;" data-ke-size="size26">GPU (Graphics Processing Unit, 그래픽처리장치)</h2>
<p style="list-style-type: none; color: #000000;" data-ke-size="size16">GPU는 원래<span>&nbsp;</span><b>그래픽과 이미지 렌더링</b>을 위해 설계된 장치로, 최근에는 AI, 머신러닝, 데이터 분석 등<span>&nbsp;</span><b>병렬 연산이 필요한 분야</b>에서도 활용.<span>&nbsp;</span><span style="background-color: #ffffff;" data-priority="2" aria-hidden="true"></span></p>
<ul style="list-style-type: disc;" data-ke-list-type="disc">
<li style="list-style-type: disc;"><b>주요 특징</b>: 수백~수천 개의 작은 코어로 구성되어 동시에 많은 데이터 처리 가능.<span></span></li>
<li style="list-style-type: disc;"><b>활용 분야</b>: 3D 그래픽, 게임, 영상 편집, 딥러닝 모델 학습, 고해상도 영상 렌더링 등.</li>
<li style="list-style-type: disc;"><b>연산 방식</b>: 반복적이고 대량의 계산을 동시에 처리하는 데 최적화되어 있으며, 부동소수점 연산에 강점.</li>
</ul>
<h2 style="list-style-type: none;" data-ke-size="size26">CPU와 GPU의 차이</h2>
<ul style="list-style-type: disc;" data-ke-list-type="disc">
<li style="list-style-type: disc;"><b>코어 구조</b>: CPU는 소수의 강력한 코어, GPU는 수천 개의 단순 코어로 구성.</li>
<li style="list-style-type: disc;"><b>연산 방식</b>: CPU는 순차적이고 복잡한 연산에, GPU는 병렬적이고 반복적인 연산에 최적화.</li>
<li style="list-style-type: disc;"><b>역할 비유</b>: CPU는 &lsquo;관리자&rsquo;처럼 하나씩 정교하게 처리, GPU는 &lsquo;실행가&rsquo;처럼 동시에 많은 작업을 처리<span></span>.</li>
<li style="list-style-type: disc;"><b>하드웨어 구성</b>: GPU는 그래픽카드 안에 포함되며, 메모리(VRAM), 냉각 장치, 전원부와 함께 완성품으로 제공.<span style="background-color: #ffffff;" data-priority="2" aria-hidden="true"></span><span></span></li>
</ul>
<h2 style="list-style-type: none;" data-ke-size="size26">요약</h2>
<p style="list-style-type: none; color: #000000;" data-ke-size="size16">CPU와 GPU는 서로<span>&nbsp;</span><b>보완적인 관계</b>에 있으며, CPU는 논리적이고 복잡한 연산을, GPU는 반복적이고 대량의 연산을 담당. 컴퓨터 성능을 최적화하려면 두 장치의 역할과 특성을 이해하고, 작업 유형에 맞게 활용하는 것이 중요.</p>
